# Session Start Prompt

Use this at the start of a future AI coding session:

1. Read `AGENTS.md`.
2. Read `ai/context.md`, `ai/current-state.md`, `ai/task-map.md`, `ai/open-threads.md`, and `ai/tasks/next-actions.md`.
3. Skim recent `ai/session-log.md` entries.
4. Run or inspect `git status --short`.
5. State:
   - current repo state,
   - relevant open threads,
   - intended change,
   - checks likely to run.
6. Treat `AGENTS.md`, `ai/`, and `docs/decisions/` as Codex-managed continuity files.
7. Only then begin code changes.
