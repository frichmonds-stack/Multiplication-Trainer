# Project Notes

This file is for feature ideas, design notes, and backlog items that should not get lost between sessions.

## Feature Backlog

### Flex Mode

- Add a competitive mode called `Flex`.
- This should be a score-chasing / leaderboard-style mode rather than a standard practice workout.
- Possible variants:
  - Points-based scoring where players compete for the highest score.
  - A pressure-timer variant with a very short countdown that resets after each correct answer.
  - If the timer runs out, the run ends immediately.
  - The core challenge is chaining the longest streak of correct answers under pressure.
- Overall design direction:
  - `Flex` modes should feel competitive rather than instructional.
  - They should reward speed, consistency, and streak-building.
